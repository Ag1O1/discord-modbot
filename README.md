# Discord Mod Bot

> [!Note] This project is partially inspired by honey pot

## Purpose

This is mainly a bot to catch spambots but I may also add general moderation
commands too

New scam bots have been coming up the past few weeks. And so I wanted to write a
bot that can catch these scam bots and ban them similarly to honeypot.
Reason I didn't just use honey pot is because I wanted to have something I can
modify easily and add features to as needed.
This bot was made specifically for a server which I moderate.
